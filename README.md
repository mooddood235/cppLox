# cppLox

C++ implementation of the Lox language described in the book [Crafting Interpreters](https://craftinginterpreters.com/) by Robert Nystrom. 

## Features
- Dynamic Typing
- Lexical Scoping
- Variables
- First-Class Functions
- Control Flow
  - Conditionals
  - For Loops
  - While Loops
- Classes
  - Inheritance
  - Polymorphism
  - Methods

## Implementation
- Recursive Descent Parser
- Tree Walk Interpreter
